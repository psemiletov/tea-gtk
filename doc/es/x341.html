9.4. Ortografía

Ortografía - TEA utiliza Aspell para la revisión ortográfica. En este menú se puede encontrar la lista de diccionarios instalados de aspell (de la distribución de linux o similares). Afecta a todo el documento, no sólo a la selección.

Las palabras incorrectas se pueden marcar con un color diferente definiéndolo en Opciones > Colores.

TEA sólo muestra los errores. El usuario ha de corregirlos, a mano. Cuando se corrige un error la palabra no cambia de color. Esto es así debido al comportamiento que sigue TEA - actualiza manualmente, por lo que para actualizar el resaltado hay que activar el menú ortográfico cada vez. Para desactivar el resaltado de errores, usar Ver > Ocultar resaltado.
