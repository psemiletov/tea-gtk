8. Etiquetas

8.3. Elementos comunes

Algunas de las funciones que se nombran a continuación funcionan de dos formas posibles - si no hay texto seleccionado se insertará una pareja de etiquetas en el documento. En caso contrario, el texto se colocará entre ellas.

Cerrar la etiqueta actual - con esta opción se cierra la etiqueta actual. Se escribe cualquier etiqueta con ">" en el final, y luego se utiliza esta función del menú. Otra forma es activar el completado automático en Opciones > Editor > Cerrar automáticamente etiquetas. Estas utilidades son capaces de trabajar con Docbook, HTML etc.

Comentario - encierra la selección actual entre etiquetas de comentarios. Funciona con TeX, HTML, CSS, Pascal y C/C++, siguiendo las reglas de sintaxis.

Color - abre el diálogo de selección de color. Si no se selecciona texto, el color HTML lo coloca en la posición actual del cursor, y en caso contrario el texto se cerrará entre etiquetas de span con la propiedad de color de las hojas de estilo CSS.

Imagen - muestra el diálogo de abrir imagen. Aquí se puede elegir la imagen[es] que se vayan a insertar entre etiquetas img (junto a atributos tales como alto y ancho) u objectos imagen (Docbook) en el documento activo. Los formatos admitidos son todos los de GTK+ 2 - PNG, JPEG, GIF, WBMP etc.

Tablas > Construir tabla - No me gustan los asistentes, pero siempre he querido una herramienta que me permitiese crear tablas vacías rápidamente. Y aquí está. De nuevo aparece en el escenario la Famosa entrada de texto. Es muy sencillo. Sólo hay que teclear: nº filas~nº columnas, y aplicar esta función. Un ejemplo, para crear una tabla de dimensiones 2x4 (2 filas, con 4 celdas en cada fila) la plantilla necesaria sería: 2~4
