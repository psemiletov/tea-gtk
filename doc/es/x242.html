8. Etiquetas

8.2. Herramientas HTML

Tamaño del documento - calcula el tamaño total del archivo incluyendo todos los elementos SRC (imágenes, flash, etc). El resultado se muestra en el registro de sucesos.

Plantilla interna - inserta la plantilla HTML en el documento actual (HTML 4.01 Transitional - EN).

Poner enlace seleccionado entre etiquetas - si se ha de poner una dirección www.holahola.bar o ftp.leech.bar, o mailto:datte-bayo@bar.com con una etiqueta a href , esta es la función adecuada.

Recortar etiquetas - elimina todas las etiquetas del documento HTML.

Convertir etiquetas en HTML - en caso de tener que mostrar código HTML con etiquetas en un documento HTML, se selecciona dicho código y se le aplica esta función. Un ejemplo sería:

&lt;b&gt;demo&lt;/b&gt;

Formateado - esta opción convierte un archivo de texto plano en código HTML con formato utilizando espacios de no separación (nbsp) para los márgenes de párrafos, etiquetas BR y otros elementos poco "ortodoxos". El resultado no está mal.
