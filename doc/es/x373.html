9.7. Recortes

Un recorte es un trozo de código que se puede insertar en un documento. TEA guarda cada recorte en un archivo por separado en $HOME/.tea/snippets . El nombre de archivo de un recorte es un elemento del menú Recortes, es decir el contenido del menú de Recortes es la lista de archivos que hay en $HOME/.tea/snippets. Para crear uno nuevo:

1. Escribir algún texto.

2. Ir a Archivo > Guardar (Especial) > Guardar como recorte.

3. Disfrutar :)

Los recortes sólo se pueden guardar en UTF-8

Se puede crear un recorte que modifique un texto seleccionado de cierta forma. Por ejemplo, se puede crear un recorte que encierre el texto seleccionado entre unas etiquetas HTML. Si %s representa ese texto. Este sería el código del recorte:

<a href="%s">%s</a>

Cuando se aplique este recorte, %s será sustituido por el texto seleccionado. Si no hubiera algo seleccinado, el contenido del recorte se incrustaría en el documento.
