10. LaTeX

¿Cómo editar documentos de TeX/LaTeX en TEA? Primero, hay que ajustar los valores en Opciones > Comandos > LaTeX. Son correctos por defecto, pero conviene comprobarlo. Y NO añadir el caracter & después de los comandos. Es necesario tener instalados todos los componentes de LaTeX.

Para exportar un documento LaTeX a dvi, utilizar la función Procesar con LaTeX y comprobar el registro de sucesos por si aparecen errores. Si todo ha ido bien, convertir dvi a ps o PDF. Para ello usar las funciones DVI a PS o DVI a PDF. TEA guarda todos los dvi's y archivos convertidos en el directorio en el que se guardó el documento LaTeX. TEA se encargará de renombrar los archivos, no es ningún problema.

Para ver un archivo dvi, se puede utilizar la opción del menú Ver dvi. Para ver archivos PostScript y PDF están las funciones Ver PS y Ver PDF.

Resumiendo, el método para generar un PDF de un archivo LaTeX con TEA es: Procesar con LaTeX, luego convertir dvi a PDF, y Ver PDF para visualizar el resultado.

O, una manera alternativa es, si se tiene instalada la utilidad pdflatex, usar Procesar con pdflatex y luego Ver PDF.
