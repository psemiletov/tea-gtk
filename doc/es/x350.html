9.5. UNIversal Text AnalyZer

¿Qué es UNITAZ? Es una herramienta precisa para el análisis de textos. Puede mostrar la lista de palabras de un documento, y el número de veces que aparece cada una.

Invocar UNITAZ con orden alfabético - hace un análisis y devuelve la lista de palabras en ese orden,

Invocar UNITAZ (sin parámetros) - realiza el análisis sin seguir un orden,

Invocar UNITAZ con orden numérico - lo mismo, pero ordenando según las veces que aparece una palabra (de mayor a menor).

Extraer palabras - extrae todas las palabras del documento. Se puede emplear, por ejemplo, para hacer un diccionario.
