Chapter 8. Markup

8.1. About all that stuff

Let me say some words about Markup menu. It can act in the several modes according to the mode choosed from the Markup mode menu. There are HTML, XHTML, Wikipedia, LaTeX and Docbook modes are available. But! Not all menu items of the Markup menu supports all markup modes. For example, the Make a table function can create tables with HTML, Wikipedia and Docbook rules, but not a TeX table.

TEA can switch the Markup mode automatically, just turn on the Preferences > Switches > Automatic markup mode option.
