9.11. Tools

Mass encode tool - use this too to encode multiply files from the one encoding to another. Optionally, you can turn on the Convert ends of lines option and choose LF or CR/LF mode.

Take screenshot (screen/region/window/clipboard). Using this functions, you can make screenshots those will be saved at the current file's directory. So to make a screenshot, open the text file first. Then, enter the file name (without a fullpath, i.e. "test.png" but not "/mnt/home/test/test.png") into the Famous Text Entry. Then use one of those functions to make a screenshot.

How to choose the image format? TEA determines it by the file's extension. You may use "png" (TEA uses the full-color PNG for file saving), "jpg", "jpeg", "bmp", "tif" and "tiff". For example: "test.jpg". To set the JPEG compression level, use the Preferences > Functions > JPEG compression (1..100) option. To change the delay before the grabbing an image, use Preferences > Functions > Screenshot delay (in seconds). During this time, TEA's main window will be hidden and you shall be able to manipulate with other windows/desktop.

Some words about a taking of "screenshot" from the Clipboard. As I tested it, it works fine with GIMP. You can select an area in GIMP, then copy it, and then go to TEA and grab the screenshot from the clipboard.
