13.6. Editor

Smart Home/End - if enabled, the cursor moves to the paragraph's start/ending.

Tab size in spaces - defines the default tab size, in the count of characters, so, 3 = approx. 3 spaces.

Enable the autosaving - yes, it is. But you need to define the Autosaving interval, in minutes. The changes will be applied after you shall restart TEA.
